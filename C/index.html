double x;		 指数

ctype.h

函数名称 函数定义 函数功能 说明
isalpha int isalpha(ch)
char ch;		 是否字母 如果是，返回1；否则，返回0；
isdigit 是否数字
isxdigit 是否十六进制数字
isgraph 是否可打印，ASCII值在0x21-0x7e之间，不包括空格
isprint 是否可打印，ASCII值在0x21-0x7e之间。
iscntrl 是否控制字符
islower 是否小写字母：a-z
isupper 是否大写字母：A-Z
tolower int tolower(ch)
int ch;		 将ch转换为小写字母返回
toupper int toupper(ch)
int ch;		 将ch转换为大写字母返回
ispunct 是否符号：除去字母，数字，空格外的可打印字符
isspace 是否是空格/制表符/换行符

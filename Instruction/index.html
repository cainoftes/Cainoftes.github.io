数据运算指令

数据传送指令

  1. 通用数据传送指令
指令 格式 作用
Mov Mov DST, SRC 将SRC的数复制到DST中
MOVSX MOVSX DST, SRC 将SRC的数复制到DST中，空出的位补SRC的最高位（SRC的位数可能比DST小）
MOVZX MOVZX DST, SRC 将SRC的数复制到DST中，空出的位补0
PUSH PUSH SRC 先修改堆栈指针，然后把SRC压入堆栈单元
POP POP DST 把堆栈指针所指向的单元弹出到DST，然后修改堆栈指针指向新的栈顶单元
XCHG XCHG OPR1, OPR2 交换两个操作数
  1. 输入/输出指令
指令 格式 作用
IN IN ACR, PORT 将PORT的内容传给ACR
OUT OUT PORT, ACR 将累加器ACR的内容传送给PORT
  1. 地址传送指令
指令 格式 作用
LEA LEA REG, SRC 将原操作数的有效地址送给寄存器
  1. 标志传送指令
指令 格式 作用
PUSHF PUSHF 先将SP的值减2，然后压入FLAGS的内容
POPF POPF 将堆栈指针的内容弹出到FLAGS，然后SP的值加2
PUSHFD PUSHFD 先将SP的值减4，然后压入EFLAGS的内容
POPFD POPFD 将堆栈指针的内容弹出到EFLAGS，然后SP的值加4

算术运算指令

  1. 类型转换指令
指令 格式 作用
CBW CBW 将AL寄存器中的符号位扩展到AH中
CBD CBD 将AX寄存器中的符号位扩展到DX中
CDQ CDQ 将EAX寄存器中的符号位扩展到EDX中
CWDE CWDE 将AX寄存器中的符号位扩展到EAX的高16位中
  1. 二进制加法指令
指令 格式 作用
ADD ADD DST, SRC 将DST与SRC的和保存到DST中
ADC ADC DST, SRC 将DST与SRC与CF的和保存到DST中
INC INC DST 将DST中的值自加1
XADD XADD DST, SRC 假设初始记为1，运算后记为2；
SRC2=DST1 ;DST2 = DST1 + SRC1
  1. 二进制减法指令
指令 格式 作用
SUB SUB DST, SRC 将DST-SRC的值保存到DST中
SBB SBB DST, SRC 将DST-SRC-CF的值保存到DST中
DEC DEC DST 将DST中的值自减一
CMP CMP DST, SRC 做减法，改变FLAGS，但是不改变DST
NEG NEG DST 对DST的所有位取反加一
  1. 二进制乘法指令
指令 格式 作用
MUL MUL SRC$_{reg/m}$ 将SRC作为乘数，将EAX作为被乘数，结果保留在EDX:EAX中
IMUL IMUL SRC$_{reg/m}$
  1. 二进制除法指令
指令 格式 作用
DIV DIV SRC$_{reg/m}$ 将EDX:EAX作为被除数，SRC作为除数，结果保留在EAX中，余数保留在EDX中
IDIV

位运算指令

  1. 逻辑运算指令
指令 格式 作用
NOT NOT DST
AND AND DST, SRC
OR OR DST, SRC
XOR XOR DST, SRC
TEST TEST OPR1, OPR2
  1. 位测试指令
指令 格式 作用
BT BT DST, SRC 测试由SRC指定的DST中的位（送给CF）
BTS BTS DST, SRC 测试并置一
BTR BTR DST, SRC 测试并清零
BTC BTC DST, SRC 测试并取反
  1. 基本移位指令
指令 格式 作用
SHL SHL DST, CNT
SHR
SAL
SAR
  1. 循环移位指令
指令 格式 说明
ROL ROL DST, CNT
ROR ROR DST, CNT
RCL RCL DST, CNT C 代表Carry而不是Circle
RCR RCR DST, CNT
